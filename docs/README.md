## 文档编写规则

- 真实代码比 Markdown 更优先
- 每个公开示例都必须映射到 `examples/`、`test/` 或明确标记为说明性
- benchmark 页只在有真实命令与结果时发布性能结论
- 脚本说明必须与 `scripts/` 的真实行为一致
